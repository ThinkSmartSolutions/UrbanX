# UrbanX Branding Pack

## Pentru GitHub Pages la `/urbanx/`
Copiază toate fișierele din acest folder în root-ul repo-ului UrbanX și pune conținutul din `urbanx-head-snippet.html` în `<head>`.

## Pentru domeniu root
Folosește `urbanx-head-snippet-root.html`.

## Fișier principal
`favicon.ico` include mai multe dimensiuni: 16, 32, 48, 64, 128 și 256 px.
